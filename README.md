# target-tracking
MTTrack 是一个模块化的多目标实时追踪系统，结合了 YOLO 目标检测、ByteTrack/SORT    追踪算法以及视觉语言模型（VL Model）进行目标分类。    该系统采用分层架构设计，分为领域层（核心追踪算法）、基础设施层（YOLO   检测器、VLLM   客户端、视频读写）、服务层（业务逻辑编排）和接口层（命令行工具）。支持   ByteTrack 和 SORT 两种追踪算法，可与任意 YOLO 模型集成，并提供可选的 VL   模型分类功能来增强目标识别能力。    通过简单的命令行即可完成视频处理，自动生成带边界框、追踪 ID   和类别标签的输出视频。代码结构清晰，易于扩展和维护，适合用于视频分析、智能监控   、自动驾驶等多种场景。
